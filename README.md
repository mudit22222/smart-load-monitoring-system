# Smart Electrical Load Monitoring System

## 🚀 Overview
IoT-based system using ESP32 to monitor real-time electrical load.

## ⚡ Features
- Real-time voltage and current monitoring
- Data logging using SQL
- Dashboard visualization

## 🛠️ Tech Stack
- ESP32, Embedded C
- Python, SQL
